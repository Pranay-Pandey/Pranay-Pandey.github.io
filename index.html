  • Self Driving Car

    Machine Learning

    View on Github

    This project focuses on autonomous car navigation using a Pixy camera, developed in a ROS2 environment with Gazebo simulation on Ubuntu 20.04. The car navigates complex tracks, including bridges and zigzag paths, relying solely on computer vision. The nxp_track_vision.py script extracts vectors from the Pixy camera to identify track boundaries, while aim_line_follow.py controls the car's steering and movement. The car adjusts its steering based on pixel density analysis, where higher white pixel density on one side causes the car to steer in that direction. A PID controller ensures smooth turns. Velocity adjustments are made during uphill and downhill sections, using odom feedback to maintain consistent speed. This project showcases advanced autonomous navigation, combining computer vision and ROS2 for complex environments.

  • Automatic conveyer barcode detection

    Machine Learning

    View on Github

    The BarCode_Reader project detects barcodes on grocery items in images using OpenCV, Zbar, and YOLO. The process involves loading the image, applying YOLO object detection, and using Pyzbar and Dynamsoft barcode readers to identify barcodes on cropped items.
